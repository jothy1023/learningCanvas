### learningCanvas
+ 项目简介

本页面是一个简单的交互页面。页面中有一张图片，鼠标移入图片时，图片上渐显出现闪烁的星星，鼠标移出星星渐隐。
+ 使用到的技术
  - CANVAS
  - Javascript

+ 项目重点难点
  - requestAnimFrame方法
    根据电脑配置选择合适的setTimeout速度
  - gameloop 方法
    调用requestAnimFrame方法实现定时渲染页面
  - Javascript的类
    由画一个星星到画多个星星的实现。利用js的类，新增原型方法与属性，生成多个实例。
  - CANVAS的save、restore方法的配套使用，以及globalAlpha方法调整透明度
